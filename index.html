Fala man, qual a bronca? Hoje estou como aluno de ciência da computação no Centro de informática (CIn), na Universidade Federal de Pernambuco (UFPE), Brasil. Gosto de me desafiar com vários projetos sozinho durante as férias (ainda mais no tempo pandêmico ocorrido em 2020 E.C, que é o ano 0 PP, pós pandemia) para aprender e praticar as mais diversas tecnologias.

Além disso venho cultivando hobbies como a prática de desenhos (mas não sei pintar), games (apesar de jogar muito mal), violino e claro uma boa leitura (de assuntos relevantes, não me venha com livros da cultura pop). Entretanto conheço muito pouco sobre filmes, séries, animes e animações, ainda assim gostando de assistir um bom vídeo no youtube ou um curso online.

Alguns dos meus projetos

  • Bible Comment

    Este é o meu projeto de fim de semanas do ano pandemico, no qual a motivação inicial foi de armazenar e compartilhar os meus próprios comentários biblicos. É o primeiro projeto React sozinho, me fazendo praticar essa questão além da conexão com MySql e certas CSS tricks.

    Tive que criar o backend usando Express, mas não encontrei a tradução da bíblia que eu queria, então eu fiz um scrapy de um site para tal, além disso quebrei muita cabeça com deploy (heroku e netlify), migrations e domínio, mas valeu a pena.

    É o projeto que mais me preocupei com o visual, justamente por que meus amigos podem se beneficiar desse projeto, então sempre surge uma nova ideia, me fazendo aprender também que como o Python, o React tem várias soluções desenvolvidas pela comunidade sem a necessidade de inventar a roda.

    BibleComment

  • IQOption Bot

    Este foi o projeto desenvolvido durante a pandemia, no qual se constitui como um automatizador de trades na plataforma IQ Option. Além de ter sido um grande passo em questão de interface desktop, se tornou o maior projeto escrito por mim até o fim de 2020 (sendo mais de 3k de linhas juntando o bot, bot do telegram, aplicação desktop entre outras coisas).

    Utilizei meus conhecimentos de várias áreas, no qual é um bot do automatizador, um bot no telegram com botões, um banco de dados usando MongoDB para a computação de licenças e tudo isso rodando no Google Cloud.

    Foi o primeiro projeto usando computação em núvem, no qual existe um script responsável pelo telegram que irá ser a interface do usuário, que é o mesmo que quando iniciado uma operação irá instanciar novas máquinas mantendo 5 usuários por máquina.

    IQOption Bot

  • Pacman em Java

    Este foi o segundo jogo feito, com o objetivo primário de ser apresentado como projeto da disciplina de introdução a programação, sendo realizado em equipe, porém acabou que tive que fazer sozinho durante as férias.

    Um projeto interessante para se conhecer um pouco de como criar uma interface gráfica em java, além de criar uma estratégia diferente para cada inimigo, se utilizando de conhecimento de algorítmos como DFS e BFS.

    Por fim foi interessante a questão de mapear imagens para os movimentos de cada personagem e a compilação o programa final podendo ser executado sem a necessidade do JDK.

  • Tkinter Game

    Este projeto foi o meu primeiro game, criado entretanto sem usar uma biblioteca especifica para jogos (como o pygame). Por outro lado foi usado uma biblioteca para interface gráfica (tkinter). A motivação inicial do projeto foi aprender orientação a objetos antes de fazer a cadeira de estrutura de dados, mas acabou indo além, onde uma parte interessante foi desenvolver a rotina dos inimigos me fazendo sempre perder. Além disso utilizei o que aprendi na cadeira de redes para adicionar um multiplayer online além do local.

    Tkinter Game

  • TodoTxt puro

    Este projeto por sua vez foi feito para treinar e aprender como desenvolver alguma aplicação usando HTML, CSS e JavaScript puro.

    Nele foi treinado vários conceitos do CSS que ainda não sabia fazer, como uma section fixada ou sobreposta as demais. Assim como implementações javascript + html, como o drag and drop e DOM.

    TodoTxT puro

  • LCS para Diff

    Não sou alguém muito interessado com competições. Mas entendo a importância de exercitar a mente para desenvolver algorítmos bem otimizados, dessa forma me interessei como acontece a comparação de commits no git, dessa forma corri atrás para aprender esses algorítmos.

    Além do mais, na interface gráfica ousei por curiosidade usando Eel que é uma biblioteca que liga o script python com a web, podendo criar a interface usando HTML normal. E em outra página usei Brython que seria usar o Python como se fosse Javascript podendo renderizar até mesmo tags HTMl.

    Diff algorithm
